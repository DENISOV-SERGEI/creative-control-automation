# Инструкция по эксплуатации автоматизации

## 1. Добавление новых креативов
Откройте таблицу `creatives_monitor`. Добавьте строку:
- `id` – уникальный номер
- `Title` – название акции
- `End_date` – дата в формате `ДД.ММ.ГГГГ`
- `Active` – поставьте `TRUE` (или галочку)
- `log` – оставьте пустым

## 2. Изменение расписания
В n8n откройте основной workflow → узел `11:00 Trigger` → измените Hour / Minute → Save.

## 3. Что делать при ошибке
- Вы получите Telegram-сообщение с текстом ошибки и названием узла.
- Зайдите в n8n → Executions → найдите красную запись → посмотрите детали.
- Исправьте причину (проверьте таблицу, лист, формат дат, доступ к API).
- Перезапустите workflow вручную для проверки.

## 4. Ручной тест
- Добавьте тестовый креатив с датой вчера.
- Запустите workflow вручную.
- Проверьте: active=FALSE, log заполнен, пришло Telegram, запись в Audit Log.
- Удалите тестовый креатив.
